GPU Acceleration

Video processing uses GPU acceleration when available:

  • Intel (QSV): h264_qsv, hevc_qsv
  • AMD/Intel (VAAPI): h264_vaapi, hevc_vaapi
  • CPU Fallback: Software decoding
