Media Service Web UI

A web interface for the Media Service microservice that preprocesses multimedia files for LLM consumption.

Supported Formats

Images: JPEG, PNG, WebP, AVIF, GIF, RAW, HEIC
Audio: MP3, WAV, OGG, M4A, FLAC
Video: MP4, MOV, MKV, AVI
